Nível de Experiência

Tag select - Atributos

  • name: Define o nome do campo enviado ao servidor com os dados selecionados.
  • required: Torna obrigatória a seleção de pelo menos uma opção.
  • size: Define o número de opções visíveis na lista sem rolagem (padrão é 1).
  • autofocus: Faz o elemento receber foco automaticamente ao carregar a página.
  • form: Associa o select a um formulário específico pelo `id` do formulário.
  • multiple: Permite selecionar várias opções simultaneamente.

Tag option - Atributos

  • value: Define o valor enviado ao servidor quando a opção é selecionada.
  • selected: Faz com que a opção esteja pré-selecionada ao carregar a página.
  • disabled: Impede que a opção seja selecionada.
  • label: Define um texto alternativo para exibição na interface, diferente do texto interno ou do valor.

Nível de Experiência

Tag optgroup - Atributos

  • label: Define o texto descritivo do grupo, exibido como título na lista suspensa (obrigatório).
  • disabled: Desativa todas as opções dentro do grupo, tornando-as não selecionáveis.
