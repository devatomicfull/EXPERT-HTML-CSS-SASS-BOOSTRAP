a tag de imagem img é usada para exibir a imagem em uma página da web a tag img tem atributos: src: caminho/url ou imagem.extensao
alt : descreve o que é a imagem caso não possa renderizar a imagem é uma alternativa
width: largura e height: altura
as imagens não são inseridas nas páginas da web, elas são mais como links para as páginas da web
o texto alternativo é considerado muito útil da perspectiva de SEO

SEO (Search Engine Optimization) em HTML refere-se à aplicação de técnicas de otimização para mecanismos de busca, utilizando a linguagem de marcação HTML, para melhorar a visibilidade e o ranqueamento de um site nos resultados de pesquisa. Isso implica otimizar a estrutura do site, as tags HTML e o conteúdo para que sejam compreendidos e indexados corretamente pelos motores de busca, como o Google.

Sobre caminho relativo e caminho absoluto

O ponto (./) refere-se à pasta atual e o restante do caminho permanece o mesmo. Independentemente de onde você colocar seus arquivos, ele se referirá à pasta atual. O navegador não terá problemas para acessar os arquivos quando o caminho for alterado.
