tag a/a é usado para definir um hiperlink, que liga de uma página para outra.
atributo href da tag a : href:" link/caminho/endereço/url"
o atributo target especifica onde abrir o documento vinculado

valores possiveis:

  • _blank:Abre o documento em uma nova janela ou aba
  • _parent:Abre o documento no quadro pai
  • _self:Padrão. Abre o documento na mesma janela/guia em que foi clicado
  • _top:Abre o documento no corpo inteiro da janela
hypertext reference: (Adress/Caminho/Url da pagina web ou arquivo)

# em um href refere-se ao ID exclusivo do elemento HTML da janela do navegador que deve ser rolada até o ID.

tag a href="#link" / nome do id /a --> id=#teste
tag a div id="link" /div
tag href="#" vamos para o topo /a ---> teste
o href="#" não especifica nenhum nome de id, mas tem um local correspondente na página - que é o topo da página
logo, quando não há um id para segmentar, ele vai para o topo da página
