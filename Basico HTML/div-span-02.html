Qual é a diferença entre div e span?
div é chamado como um elemento de bloco (refere-se a um tipo de elemento que ocupa toda a largura disponível na página e começa em uma nova linha) , enquanto span é chamado como um elemento inline:
(forma de apresentar elementos que fluem dentro do texto principal, ocupando apenas o espaço necessário para o seu conteúdo).
