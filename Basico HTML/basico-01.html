  • tag html/html: Não são visíveis para o leitor no navegador, eles estão lá, mas você não pode vê-los
  • tag head/head (cabeca): onde fica as tags (metadata), lembrando que metadados é uma coleção de dados que descrevem os dados mas voltando a tag head, são os dados sobre as informações reais da página
  • tag title: Nada mais é do que o titulo da pagina, veja voce mesmo ao abrir essa pagina no navegador
  • tag body/body (corpo): Corpo da Pagina é onde fica o conteudo visível para o leitor
  • Podemos definir outras tags detro do corpo

Mas afinal o que é a tag e como é sua sintaxe? A tag é uma fora de falar para o navegador o que deve ser mostrado e como deve ser mostrado.
exemplo a tag p diz ao navegador que é um paragrafo, logo o navegador vai formata como paragrafo.
Html é uma linguagem de marcação.
sintaxe:
Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final:
< tagname> O conteúdo vai aqui... < /tagname>
O elemento HTML é tudo, desde a tag inicial até a tag final.

o Html segue principio (Estrutura de Arvore) com suas tags:
html tag raiz (PAI)
2 tags FILHAS: head e body
a tag filha do head neste exemplo seria title

ja a tag filha de body poderia ser muitas outras

tag p/p (PARAGRAFO): Esta tag pode conter textos, mensagens, imagens, links(urls) e muitas outras tags
Ela tem uma margem de espaço sendo espaco em cima e embaixo, logo ao usar vai perceber um unica linha de espaço fazendo a divisão
Tem seu próprio conjunto de atributos sobre como ele vai ficar
Espaços EXTRAS e linhas extras serão considerados como um único espaço e serão removidos quando exibidos no navegador

algumas tag não necessitam de fechamento por exemplo a tag hr que faz uma linha na pagina você pode colocar apenasno formato de tag aberta ou fechar no final da tag aberta por exemplo hr/

Mas como boa pratica é acoselhado a fechar no final da tag aberta
a tag que não tem conteúdo e não precisa de uma tag de fechamento/finalização é chamada de tag vazia

atributos:
Os atributos HTML fornecem informações adicionais sobre elementos HTML.

  • Todos os elementos HTML podem ter atributos
  • Os atributos fornecem informações adicionais sobre os elementos
  • Os atributos são sempre especificados na tag inicial
  • Os atributos geralmente vêm em pares nome/valor como: nome="valor"
exemplo: o atributo style que podemos definir o estilo da nossa tag style="nome:valor"
style="color=blue" existe muito outros atributos como : id, class, src, href, alt ...
