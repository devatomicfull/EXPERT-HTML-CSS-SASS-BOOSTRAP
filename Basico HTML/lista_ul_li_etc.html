As tags de lista são usadas para exibir as informações na forma de listas

Lista Não Ordenada

ul é pai de li

Nao é ordernado o formato logo voce vai ver bolinhas
ul -> define como lista
li -> define o item da lista, ou melhor, lista o item.
o li é necessário para escrever os itens da lista

  • item 1
  • item 2
  • item 3
  • item 4

Existe outros tipos alem da bolinha, como: circle, square, none(nenhum), disc (padrão) que é o atual.

  • item novo
  • item velho

Lista Ordenada

o ol é usado para exibir os itens da lista em formato ordenado

ol é pai de li
  1. ordem
  2. ordem
  3. ordem
diferentes tipos de números estão disponíveis, numeral, romano, letras i, alfabeto maiusculo ou minusculo
também podemos exibir os itens a partir de um determinado número/alfabético
  1. ordem
  2. ordem
  3. ordem

lista em números romanos minúsculos começando pelo número 6, ou seja, "vi"

  1. teste
  2. teste
  3. teste
número/alfabético: os valores podem ser exibidos em ordem inversa.
  1. teste
  2. teste
  3. teste
O atributo reverso não alterará o conteúdo na ordem inversa, mas apenas os valores numéricos/alfabéticos
você também pode criar uma lista aninhada usando as listas
