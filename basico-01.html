    O html dentro do DOCTYPE NÃO É UMA TAG propriamente dita, mas um comentário para o navegador sobre a versão HTML. É ESSENCIAL ESCREVER isso no topo de cada página HTML que você Criar.
  • tag html/html: Não são visíveis para o leitor no navegador, eles estão lá, mas você não pode vê-los
  • tag head/head (cabeca): onde fica as tags (metadata), lembrando que metadados é uma coleção de dados que descrevem os dados mas voltando a tag head, são os dados sobre as informações reais da página
  • tag title: Nada mais é do que o titulo da pagina, veja voce mesmo ao abrir essa pagina no navegador
  • tag body/body (corpo): Corpo da Pagina é onde fica o conteudo visível para o leitor
  • Podemos definir outras tags detro do corpo

Mas afinal o que é a tag e como é sua sintaxe? A tag é uma fora de falar para o navegador o que deve ser mostrado e como deve ser mostrado.
exemplo a tag p diz ao navegador que é um paragrafo, logo o navegador vai formata como paragrafo.
Html é uma linguagem de marcação.
sintaxe:
Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final:
< tagname > O conteúdo vai aqui... < /tagname >
O elemento HTML é tudo, desde a tag inicial até a tag final.

o Html segue principio (Estrutura de Arvore) com suas tags:
html tag raiz (PAI)
2 tags FILHAS: head e body
a tag filha do head neste exemplo seria title

ja a tag filha de body poderia ser muitas outras
